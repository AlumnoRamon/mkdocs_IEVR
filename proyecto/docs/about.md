# Acerca de esta web

Esta página es una **wiki temática** dedicada a *Inazuma Eleven: Heroes Victory Road*.  
Su objetivo es ofrecer una referencia clara y completa sobre personajes, técnicas, modos de juego y el mundo de esta entrega.

---

### ¿Para quién es esta web?  
- Fans de Inazuma Eleven.  
- Nuevos jugadores que quieran entender el sistema del juego.  

---

### Aviso
Este contenido está resumido con fines educativos y divulgativos. Todas las marcas pertenecen a Level-5.
